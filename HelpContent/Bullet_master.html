Bullet Master
Publication center
This is the center from which Bullet is issued. For example Oil and Gas directory is issued from Delhi so Delhi is the Publication Center of OAGD.
Bullet Code
This field is a text box which accept an unique identification of Bullet code from an user.It is always unique code for each Bullet.This field specifies the Bullet code.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Bullet Code Prefix and next 1 for alphabetically numeric SL. No. of Bullet Code (i.e. BU0) etc.And the User define mode-Code generate itself from the user.
Bullet Description
This field specifies the Bullet Description to use Bullet Symbol for the attractive booking Ad.We have many Symbols, to representating to the bullet description.- Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default.
Bullet Charges
This field specifies the drop down for Bullet Charges i.e. Fixed or Percentage.
Validity From and Validity Till
This field specifies the textbox for Valid from date and Valid till date. Both text box user has to input.
Remarks
This field is used to give the remarks or comments regarding the Bullet .Used in the Report.
Symbol
This field specifies the Bullet Symbol which one you used.
Font
Font is specifies the size of the text.
