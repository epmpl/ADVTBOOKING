Company Master
Company Code
This field is a text box which accept an unique code for each Company from an user.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Company Code Prefix and next 1 for alphabetically numeric SL. No. of Company Code (i.e. CO0) etc.And the User define mode-Code generate itself from the user.Used in Rate Card, Ad Booking Details and Reports.
Company Name
This field specifies the name of the Company.And In this field is stored as Supreme Advertising etc.Company Name.- Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default.
Alias
This field specifies the short name of Company.
Address
Mention the complete address of the Company with selection of Country, City etc.
Email_id
Enter the email id of the person at which communication will be done as needed.
St_Acc_No
This field specifies the Account number of the company.To Used in further Transaction and Reports.
Pan No.
This field specifies the Personal Account number for the Company user.This no. is unique to another Company user.
Registration Date
This field specifies the Business Establishment date of the Company.
