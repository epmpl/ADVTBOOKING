State Master
Country Name
This field specifies the country to which the state belongs.In lndia Country includes many states are - Uttar Pradesh,Uttarakhand,Delhi,Rajastan etc.
State Code
This field is a text box which accept an unique code for each State from an user.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for State Code Prefix and next 1 for alphabetically numeric SL. No. of State Code (i.e. ST0) etc.And the User define mode-Code generate itself from the user.
State Name
This field is a text box; specifies the name of the Particular State of the selected Country.So lndia is a Country includes many states are -Uttar Pradesh, Uttarakhand,Delhi,Rajastan etc.
Alias
This field specifies the short name of State master.
