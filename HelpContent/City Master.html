Country Name
This field specifies the name of the country in which the city resides.So lndia is a Country and then its Cities are - Saket, Mohan Nagar ,Okhala, Lajpat nagar ,Mayur vihar etc.
City Code
This field is a text box which accept an unique code for each city from an user.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for City Code Prefix and next 1 for alphabetically numeric SL. No. of City Code (i.e. CI0) etc.And the User define mode-Code generate itself from the user.
City name
This field is a text box; specifies the name of the City of the selected State or Country.So lndia is a Country includes many states.U.P. is state of india and its Cities are -Mohan Nagar, Gandhi Nagar, Turab nagar ,Raj nagar etc. .- Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
Alias
This field specifies the short name of City.
State Name
This field specifies the name of the state in which the city is present.
City STD Code
This field specifies the STD Code of the city.Only numeric input is allowed.
