Ad Category Master
Ad Type Name
This field specifies the Advertisement Type to define the different categories of Ad.
Ad Category Code
This field is a text box which accept an unique identification code from an user.It is always unique code for each Ad category.This Code is auto generated whether it is user defined.In Auto generated, First 2 characters are allotted for Ad Category Code Prefix and next 1 for alphabetically numeric SL. No. of Ad Category Code (i.e. AD0) etc.And the User define mode-Code generate itself from the user.Used in Rate Card, Ad Booking Details and Reports.
Ad Category Name
This field specifies the Advertisement Categories like Business, Computer,Education or matrimonial etc.- Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
Alias
This field specifies the short name of Adv Category.
Ad Category Days
This field is used to select the Publish Day on which the Ad Category will publish.
