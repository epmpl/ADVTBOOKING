Bg Color Master
Bg Color code
This field is a text box which accept an unique identification of Bg Color code from an user.It is always unique code for each Bg Color.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Status Code Prefix and next 1 for alphabetically numeric SL. No. of Status Code (i.e. ST0) etc.And the User define mode-Code generate itself from the user.
Bg Color Name
This field specifies the name of Background Color of the page. - Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
Publication
This field specifies the Publication Name in which the Ad is published and we have to give the Bgcolor of published Ad.
Edition
This field specifies the Edition Name in which the Ad is booked and we have to give the Bgcolor of published Ad.
Category
This field specifies the Category of the Ad.
