Brand Name
This field specifies the name of the Brand. All Brands are listed in this drop down.For example you have a car of Honda Brand.And these varient are- SX4,City,Jazz and WagonR.
Varient Code
This field is a text box which accept an unique code for each Varient from an user.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Varient Code Prefix and next 1 for alphabetically numeric SL. No. of Varient Code (i.e. VA0) etc.And the User define mode-Code generate itself from the user.
