Agency Type Master
It is used to define the classification of Agency. Some Commission is applicable according to the classifications. It may be accredited or non-accredited.
Agency Type Name
Name of the Agency Type is defined. e.g. accredited or non-accredited.
Effective From
It is the start date from which commission will be applicable.
Effective To
It is the end date at which commission will be applicable.
Commission Rate
Define the commission which would be applicable.
Commission Apply On
Select the option (Net/Gross) on which commission rate will be charged at the time of booking.
