Agency Master
Agency Master is used to open the New agencies and store the records of all agencies.
Type
User has to select the Type as it is a Parent (Main) agency or Child (Sub Agency).
Agency Name
Enter the Agency Name in this field.
Agency Code
This is the Unique code which is generated by the system or manually. every agencies have its own unique agency code.
Sub Agency Code
this code is made for the sub agencies (child). This is also the Unique code which is generated by the system or manually.
Alias
This field is used to make an alias name of the agency (pet name). this is autogenerated but editable field.
Address
Mention the complete address of the Agency.
Email ID
Enter the valid email id at which communication will be done with the agency by mails.
MRV Reference Number
Enter the MRV Reference number.
Credit Days
Mention Credit days of the Agency. it is the time period of the agency to credit its balance into the account of publication house.
Credit Limit
How much the amount is credited by any agency is defined in this field. its the max limit.
Bill To
select the Agnecy name on which billing will be done.
Representative
select the representative name.
Remark
if there is any comment to the agency then it is mentioned in this field.
Commission Details
At what commission agency is doing business with the organisation, is mentioned in this tab. you have to mention also that on which amount that commission will be applicable, net amount or gross amount.
Status Details
in status details tab, Agency status is given as it is banned Agency or active Agency. And mention the dates at which that status would be applicalbe.
Contact Details
In this tab, mention the contact person of the agency that publication house(organisation) will communicate with the agency in future. And Agency Products will be also mentioned for which agency does the booking also.
