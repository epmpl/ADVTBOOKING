Ad Size Master
Ad Type
This field specifies the Advertisement Type to define the different size of Ad.
Ad Categary
All ads are categorised specifically as regionwise and size are defined according to that specific categories.
Package
All the packages as Oil and gas Directory packages or Yellow Pages packages are displayed in this field. and Adv Size are defined here for the selected directory's packages.
Ad Size Code
This field is a text box which accept an unique identification of Ad Size code from an user.Whether this code is Auto generated or User defined.This Code is auto generated whether it is user defined. In Auto generated,First 2 characters are allotted for Ad Size Code Prefix and next 1 for alphabetically numeric SL. No. of Ad Size Code (i.e. AD0) etc.And the User define mode-Code generate itself from the user.
Ad Size Description
This field specifies the Ad Type Description. - Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
Color
Ad can be booked black & white or color and mostly both of them have different rates.
Hight
This field specifies the height of the Page.Basically that used to set the size of the page.
Width
This field specifies the width of the Page.Basically that used to set the size of the page.
UOM
All the created stylesheets are displayed here as ROL stylesheet( matter with logo ads) or DROL stylesheet(normal ads) or CD( 1-2 column image ads).
Remark
This field is used to give the remarks or comments regarding Ad Size.Used in the Report.
