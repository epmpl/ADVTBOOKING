Language Master
Language Code
This field is a text box which accept an unique code for each Language from an user.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Language Code Prefix and next 1 for alphabetically numeric SL. No. of Language Code (i.e. LA0) etc.And the User define mode-Code generate itself from the user.
Language Name
This field specifies the name of the Language in which the language are speaking an user.Mention this text box.In India includes many languages like as (Hindi,English,Marathi,Bengali and Punjabi etc.) - Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
Alias
This field specifies the short name of Language.Short name is used for printing in reports.
