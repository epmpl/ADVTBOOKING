Periodicity Code
This field is a text box which accept an unique code for each Premium from an user.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Company Code Prefix and next 1 for alphabetically numeric SL. No. of Company Code (i.e. CO0) etc.And the User define mode-Code generate itself from the user.Used in Rate Card, Ad Booking Details and Reports.
Periodicity Name
This field specifies the description of the Time Interval( Daily,Weekly,Yearly and Fornight etc). - Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default.
Alias
This field specifies the short name of Periodicity.
