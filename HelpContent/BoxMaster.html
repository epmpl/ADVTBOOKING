Box Master
Publication Center
This field specifies the Publication Center on which the box will be applicable.
Box Code
This field is a text box which accept an unique identification of Box code from an user.It is always unique code for each Box.This field specifies the Box Code of the publication.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Status Code Prefix and next 1 for alphabetically numeric SL. No. of Status Code (i.e. ST0) etc.And the User define mode-Code generate itself from the user.
Box Description
This fields communicate between the publication house and the Client,In case Client doesn't come. In that case,if he sends the documents through any box of transport like (courier,Speed Post,Postal Services) etc.Booking Ad charges based on using the box.All the box description defined in that text box. - Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
No of Dispatches
This fields specifies the user how many times to using the box.
Box Charges
This field specifies to define the Box charges for the publication.
Valid From and Valid To
This field specifies the date from the Box rate is applicable and the date up to which the selected rate is applicable.By defining the period we can identify the rate that which is applicable for the current period.
Remarks
This field is used to give the remarks or comments regarding Box.Used in the Report.
