Currency Master
Country Name
This field specifies the Name of Country name of the Currency belongs to.India belongs to rupee currency and the U.S. belongs to Doller currency.
Currency Code
This field is a text box which accept an unique code for each Currency from an user.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Currency Code Prefix and next 1 for alphabetically numeric SL. No. of Currency Code (i.e. CU0) etc.And the User define mode-Code generate itself from the user.
Currency Name
This field specifies the name of the Currency for which conversion values to be given.Many countries are using different different types of currency like as (Doller,Rupee,Pound,Dinaar etc.).- Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
Alias
This field specifies the short name of Currency.
Conversion Rate
To convert the Currency value.If we know the conversion rate.Mention this text box.
Effective From
This field specifies effective date from which the commission will be paid to Agent.To be used to calculate the due amount.
Effective Till
This field specifies the date; up to that the commission will be paid to Agent..To be used to calculate the due amount.
Commission Rate %
This field specifies the commission rate agreed with the agent.Commission rate is used to calculate the due amount of any agency.
Commission Apply on
This field specifies the commission amount to be calculated on Gross or Net. That means discount to be considered .Used to calculate the amount to be paid to agent
