Rate Master
Ad Type
There are two types of Ads. First is Classified ads in which 1 to 2 columns of ad exist. Second is Display Ads in which more than two columns ads exist.Packages, Categories and style sheets are binded with ad type.
Ad Category
There are many categories in which ads are published as Business, Property, Notice and Tender, Matrimonial etc. So user has to select category from here to define the categorywise rates.
Ad Sub Category
There can be more classifications of the main categories. For example: if Main Ad Category is selected MATRIMONIAL then its sub category can be WANTD GROOM, WANTED BRIDE. So user has to select the Respective sub category, if rates are different from selected main category. it rates are same for Main Ad CAtegory and its sub categories then there is no need to define the rates for Sub Category.
Ad Sub Sub Category
This is the Third level of ad category. it is also binded with selcond level. suppose user selects the sub category as WANTED BRIDE then its further classifications are displayed in this field as all casts are shown here.
Rate for Main Edition
When user selects this radio button then all main editions are binded in Edition Dropdown.
Rate for Center
When user selects this radio button then all sub editions and supplements are binded in Edition Dropdown.
Edition
All the Editions and supplements are shown in this drop down as HT-DEL, HT-MUM etc. These editions are displayed in this dropdown according to the selected radio button. And rates are defined here for the selected package.
Color
Ad can be booked black & white or color and mostly both of them have different rates and we can open these rates seperately.
UOM
in this field we selects the unit of measurement as ROL (Run on Linage), ROW (Run on Words), SQCM (Squre centimeter) etc. All UOMs have different rates and these rates are applied at the time of booking.
Minimum Lines
There are some measurements to fetch the rates as according to lines/words/square centimeters etc. User has to enter minimum lines for which the base rate will be applicable. for example: if min lines are 5 and base rate is 100 then this base rate is applicable 1 to 5 lines. if there is 6 lines ad then extra rate will be charged for 6th line instead of base rate. and For CD ad, Minimum size will be entered and rates will be charged from extra rate field.
Rate
Base rates will be defined in this field.
Extra Rate
if there are some extra line/size in any ad then that extra line/size rate is mentioned here.
Scheme Available
if there is any scheme is applicable with the rate then user has to select this scheme from this drop down.
Check box for slab rates
if rates are divided in slabs then user has to select this check box and some rows are shown that time and enter the slab rates.
Insert and Delete button
Insert and Delete button are used to add and delete the row respectively.
From Insert and To Insert
Enter the slabs of insertions in from and to insert field. for example, 1 to 5 insertions have different rates and 6 to 12 insertions have different rates then user has to enter these rates in these fields.
