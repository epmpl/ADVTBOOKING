Display Booking Master
This form is used only to book Display ads which occupies more than 1 column. For Display ads, we have to upload or create image . Ad can be of full page or half page etc. in this booking form, pagination can be done easily. all issue pages are displayed here and have the facility to check the space availability.
Agency Code & Client
There are some registered agencies and clients(reg. or walkin) those book ads. for example, Atlas is the client of Oil and Gas Directory(OAGD) who book ads.
RO Status
this field is used to make the booking Confirmed/Unconfirmed, means confirm booking is ready to pubish but unconfirm booking will not be published.
Dockit Number
dockit number is used when we save reserve or unconfirmed booking. dockit number is generated at the time of save of reserve booking.
Executive & Retainer
those person who bring the business( booking) for the organisation, are mentioned here.
Ad Details Tab
every booking has Some ad details which is stored in this tab. e.g Booking type is Normal or FMG, what is the color of the ad and in which category( classification) ad will be booked and which stylesheet is used to book that ad.
Package details Tab
in which package, ad will be booked, is to be selected in this tab. e.g. ad will be booked for OIL and GAS Directory(OAGD) or Yellow Pages Directory. multiple package booking can be handled easily in this tab and all details of this package booking is displayed in the insertion grid(below of the tab).
Page details Tab
in this tab, there are some fields as Ad Size Type ( as Front page, Back Page. when we select front page from here then height and width comes autometically),Page Premium (when we select page as FP then premium will be applicable for this), position premium (select the position on the page), ad size ( if ad is CD ad then assign Height & Width), Page Number (enter the page number on which ad will be published) etc.
Rate details Tab
in this tab, total rates (card rate, card amount, agreed rate, agreed amount etc.) is displayed here with all premiums and discounts.
Bill details Tab
in this tab, bill details is given regarding booking. As what is the bill cycle, revenue center, payment type, bill status, billable size, bill to, receipt no, trade discount, bill amount, bill remark.
Box details Tab
in this tab, if communication is done regarding booking by post or by courier etc. then some charge is applied for those. thsese details is mentioned in the box detail tab
