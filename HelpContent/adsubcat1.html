This field is a text box which accept an unique identification code from an user.It is always unique code for each Ad sub category.This Code is auto generated whether it is user defined.In Auto generated, First 2 characters are allotted for Ad Sub Category Code Prefix and next 1 for alphabetically numeric SL. No. of Ad Sub Category Code (i.e. AD0) etc.And the User define mode-Code generate itself from the user.
