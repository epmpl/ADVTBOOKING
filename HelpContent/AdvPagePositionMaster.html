Ad Page Premium code
This field is a text box which accept an unique identification Premium code from an user.It is always unique code for each Ad Page Premium.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Premium Code Prefix and next 1 for alphabetically numeric SL. No. of Premium Code (i.e. PR0) etc.And the User define mode-Code generate itself from the user.
Ad Page Premium Name
In this text box where is,the client can opt. for different types of pages to give their advestisement.The rates for various ads from page to page.Mean to say, if he opts for the front page then there will be seperate rate whereas if he opts for back page then the rates will be less of the front page. - Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default.
Ad Type
This field specifies whether this AD is Classified or Display.
Ad Categary
This field specifies the Advertisement Category on which the premium will be applicable.
Publication
This field specifies the Publication Name on which the premium will be applicable.
Ad Size description
This field specifies the size of the Ad on which premium is applicable.
Edition
This field specifies the Edition Name on which the premium will be applicable.
Suppliment
This field specifies the Supplement Name on which the premium will be applicable.
Premium
This field specifies the Premium type that the premium is in percentage or any fixed amount is charged.
Rate Group
This field specifies the Rate group of the Ad like Commercial etc.
Special Position
This field specifies the special position assigned to the Ad.
Solos or semi Solos
This field is the selection of Ad that the Ad should be in between of two pages (Solus) or on single page(Semi Solus).
Valid From and Valid To
This field specifies the date from the premium rate is applicable and the date up to which the selected rate is applicable.By defining the period we can identify the rate that which is applicable for the current period.
Premium Applicable For
This field Specifies the days in which premium is applicable.Multiple Days can be done.
