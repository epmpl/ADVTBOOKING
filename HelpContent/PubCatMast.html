Publication Center Master
Publication Center Code
This field specifies the code of Publication Center. This Code is auto generated whether it is user defined. In Auto generated,First 2 characters are allotted for Publication Center Code Prefix and next 1 for alphabetically numeric SL. No. of Publication Center Code (i.e. PU0) etc.And the User define mode-Code generate itself from the user.Used in Rate Card, Ad Booking Details and Reports.
Publication Center Name
This is the center from which edition is issued. For example Oil and Gas directory is issued from Dubai so Dubai is the Publication Center of OAGD.- Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
Alias
This field specifies the short name of Publication Center.
Address
Mention the complete address of the Publication Center.
Country
It specifies the name of the country in which the Publication Center is located.
Remark
This field is used to give the remarks or comments regarding Publication Center.Used in the Report.
Status Details
In status details tab, Agency status is given as it is banned Agency or active Agency. And mention the dates at which that status would be applicalbe.
Contact Details
In this tab, mention the contact person of the agency that publication house(organisation) will communicate with the agency in future. And Agency Products will be also mentioned for which agency does the booking also.
