This field is a text box which accept an unique code for each Page type from an user.Whether this code is Auto generated or User defined.In Auto generated, First 2 characters are allotted for Page Type Code Prefix and next 1 for alphabetically numeric SL. No. of Page Type Code (i.e. PA0) etc.And the User define mode-Code generate itself from the user.
Page Name

Height
This field specifies the height of the Page.Basically that used to set the size of the page.
Width
This field specifies the width of the Page.Basically that used to set the size of the page.
No. of Coloum
This field specifies the Number of Columns in the Page.
