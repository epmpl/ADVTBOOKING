Branch Master
Publication Center
Select the Publication Center for which Branch will be defined. Suppose there is a center Delhi and then its branches are South ex., Laxmi Nagar, Janak puri etc.
Branch Code
This field is a text box which accept an unique code for each Branch from an user.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Branch Code Prefix and next 1 for alphabetically numeric SL. No. of Branch Code (i.e. BR0) etc.And the User define mode-Code generate itself from the user.
Branch Name
This field is a text box; specifies the name of the branch of the selected publication center.Suppose there is a center Delhi and then its branches are South ex., Laxmi Nagar, Janak puri etc.
Alias
This field specifies the short name of Branch.
Address
This field specifies the complete address of the Branch as its country, state, city, district, pin code, email id etc.
Pin Code
This field specifies the pin code of the area of the selected city where the Branch is situated.Entry should be maximum of 12 digits and non-negative.
Contact Details
In this tab, mention all the details of contact person of the publication center which is used to communicate as needed.
