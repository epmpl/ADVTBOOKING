Scheme Master
Scheme Master is used to define the schemes which will be used at the time of booking.after creating schemes, these are autometically picked up by the system.
Scheme Name
This field specifies the Schame Name of the Package.This Name is user defined. User will have to enter scheme name manually for a special schemes and discount as 2+1 (one insertion free with two insertion), 3+2 (two insertion free with three insertion),...etc. This scheme Name can exist multiple schemes as 2+1,3+2 etc.
Insert Button
This button is used to add new row to make the one another scheme.
Delete Button
This button is used to delete the selected row.
Description
In this field, description of schemes will be defined as 2+1 or 3+2 etc.
From Insertion and To Insertion
Enter that number which has any discount or free. As for 2+1, user has to enter '3' in this text field.In To Insertion field user also has to enter '3'. Means user has to enter insertion range from which discount will be allowed.
Discount
Discount is mentioned in the percentage. It can not be greater than 100. if discount is 100% then mentioned insertions (From and To Insertion fields) will be free.
