Color Type Master
Color Type code
This field is a text box which accept an unique identification of Color Type code from an user.It is always unique code for each Color Type.Whether this code is Auto generated or User defined. In Auto generated,First 2 characters are allotted for Status Code Prefix and next 1 for alphabetically numeric SL. No. of Status Code (i.e. ST0) etc.And the User define mode-Code generate itself from the user.
Color Type Name
This fields basically used for changing the color type of the fonts. - Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default
Alias
This field specifies the short name of the color type.
Category
This field specifies the Advertisement Category on which the the Color Type is bined for the particular Category.
Edition
This field specifies the Edition Name on which the Color Type is bined for the particular edition.
