Role Code
This field is a text box which accept an unique identification number from an user.It is always unique code for each role.This Code is auto generated whether it is user defined. In Auto generated,First 2 characters are allotted for Company Code Prefix and next 1 for alphabetically numeric SL. No. of Company Code (i.e. CO0) etc.And the User define mode-Code generate itself from the user.Used in Rate Card, Ad Booking Details and Reports.
Role Name
This field specifies the name of the Role in the company.which defines the various roles is an organization.It can be Admin,Manager,Engineer etc. - Duplicate entry not allowed. - Check Mandatory while save. - Should accept values Capital Letters by default.
